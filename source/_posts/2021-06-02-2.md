# ES5 中的 typeof

## 例子

```javascript
typeof '123' // string

typeof 123 // number

typeof new Date() // object

/* typeof null 是 object 是js的bug
 * 原因是typeof判断引用地址后几位是否为0 null 是 000000...000
 */
typeof null // object

typeof (() => {}) // function
/**
 * function 是object的一种
 * 有内部call方法的object的就是方法
 */
```

## 题目

```javascript
typeof new String('abcd') // object

/**
 * new 出来的都是object
 */
```
