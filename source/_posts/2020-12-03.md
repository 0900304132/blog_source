# vue 中的style变量

  1. **css**

      ```css 
        
      --var:red|10px|100%; 
        
      color: --var; 
        
      width:--var; 

      height:--var; 

    

``` 

 2. **scss**

``` scss

--var:red|10px|100%;

color:#{--var}; 

```
